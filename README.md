# WFH-ProMax
 
A Windows application that simulates mouse movement to prevent system idle timeout. Moves the mouse left and right until a keyboard key is pressed.
 
## Prerequisites
 
- Windows
- [CMake](https://cmake.org/)

## Building
 
```bash
git clone https://github.com/E-Zian/WFH-ProMax.git
cd WFH-ProMax
cmake -B build
cmake --build build
```
 
## Running
 
```bash
./build/WFH-ProMax.exe
```
 
Press any keyboard key to stop the application.
