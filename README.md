# WFH-ProMax
 
A Windows application that simulates mouse movement to prevent system idle. Opens Gmail and moves the mouse left and right until a keyboard key is pressed.
 
## Prerequisites
 
- Windows
- [CMake](https://cmake.org/)

## Building
 
```bash
git clone https://github.com/E-Zian/WFH-ProMax.git
cd WFH-ProMax
cmake -B build
cmake --build build --config Release
```
 
## Running
 
```bash
.\build\Release\WFH_Promax.exe
```
 
Press any keyboard key to stop the application.
